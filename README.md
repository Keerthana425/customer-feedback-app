
# Customer Feedback App

## 🚀 Live Demo

https://keerthana23444.ccbp.tech/

---

## 📌 Description

A React-based customer feedback application that allows users to submit their satisfaction using emojis. Once feedback is given, a thank-you message is displayed, enhancing user interaction and experience.

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6+)
* CSS

---

## ✨ Features

* 😊 Emoji-based feedback system
* 🔁 Conditional rendering (feedback → thank you screen)
* ⚡ State management using React
* 📱 Responsive UI

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Keerthana425/customer-feedback-app.git
```

2. Navigate to project folder:

```bash
cd customer-feedback-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm start
```
